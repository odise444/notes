# Floor Planner 개발기 시리즈 기획

## 프로젝트 개요

- **목표**: 웹 기반 평면도 설계 도구
- **기술 스택**: Nuxt 3 + Vue 3 + TypeScript + vue-konva + TailwindCSS + Pinia
- **테스트**: Vitest (170개) + Playwright E2E (39개)
