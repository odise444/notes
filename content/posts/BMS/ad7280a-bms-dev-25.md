6셀 테스트 보드에서는 다 됐다. 이제 진짜 72V 팩에 연결할 차례.

솔직히 좀 무서웠다. 72V면 감전되면 위험하다.

---

안전 장비부터 챙겼다.

- 절연 장갑 (1000V 등급)
- 보안경
- 절연 공구
- 소화기 (리튬 배터리용)

![작업 환경](/imgs/box.jpg)

---

연결 순서:

1. BMS 보드 단독 전원 테스트 (5V 어댑터)
2. 통신 확인
3. 배터리 팩 전압 측정 (멀티미터)
4. GND부터 연결
5. 셀 탭 순서대로 연결 (C0 → C1 → ... → C23)
6. BMS 전원 ON
7. 셀 전압 읽기 확인

![12셀 연결](/imgs/12cells.jpg)

---

첫 전원 ON.

LED가 깜빡이고, UART로 로그가 나왔다.

```
[BMS] Init start
[AD7280A] Device 0 OK
[AD7280A] Device 1 OK
[AD7280A] Device 2 OK
[AD7280A] Device 3 OK
[BMS] All devices initialized
[BMS] Reading cells...
Cell 01: 3.198V
Cell 02: 3.201V
...
```

됐다!

---

근데 Cell 7 값이 이상했다.

```
Cell 07: 6.402V  ← ???
```

6.4V? 리튬 셀이 4.2V를 넘을 리가 없다.

확인해보니 셀 탭 커넥터가 하나 빠져있었다. C6-C7 사이가 끊어져서 두 셀 전압이 합쳐서 읽혔다.

커넥터 다시 꽂으니까 정상으로 돌아왔다.

---

교훈: 이상한 값이 나오면 배선부터 확인하자.
