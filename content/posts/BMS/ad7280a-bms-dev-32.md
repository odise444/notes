## PCB가 도착했다

택배 상자를 열었다. 반짝반짝한 PCB 10장.

**"이제 납땜만 하면 끝이네!"**

3일이면 된다고 생각했다. 실제로는 2주가 걸렸다.

---

## 첫 번째 삽질: AD7280A 납땜

AD7280A는 LQFP-64 패키지. 0.5mm 핀 피치.

```
핀 피치 비교:
┌────────────────────────────────────────────┐
│  1.27mm (DIP)      ████████  여유 있음     │
│  0.8mm (TQFP-44)   ████████  보통          │
│  0.5mm (LQFP-64)   ████████  빡세다        │
│  0.4mm (QFN)       ████████  지옥          │
└────────────────────────────────────────────┘
```

**인두기 들고 시작했다.**

첫 번째 IC. 핀 10개 정도 납땜하는데 **옆 핀이랑 브릿지.**

```
의도:               실제:
                    
█ █ █ █ █ █        █▀█▀█ █ █ █
│ │ │ │ │ │        └┬┘
                    브릿지!
```

납 흡입기로 빼고 다시. 또 브릿지. 빼고 다시.

**30분 만에 IC 패드가 벗겨졌다.** PCB 1장 폐기.

---

## 두 번째 삽질: 플럭스의 중요성

선배한테 SOS 쳤다.

**"플럭스 바르고 해."**

플럭스? 그게 뭐지? 부품함 뒤져서 찾았다. 노란 액체.

IC 핀에 플럭스 바르고 납땜했더니 **마법처럼 납이 스르륵 퍼졌다.**

```
플럭스 없이:          플럭스 있으면:
                      
납이 뭉침             납이 퍼짐
 ⊙                    ___
│ │                  │ │
브릿지 발생           깔끔
```

**플럭스의 역할:**
- 산화막 제거
- 표면 장력 낮춤
- 납의 흐름성 개선

이후로 **정밀 납땜 전에는 무조건 플럭스.**

---

## 세 번째 삽질: 열풍기 온도 설정

플럭스로 AD7280A 하나 성공. 4개 다 붙이는 데 2시간.

**"열풍기(Hot Air Gun) 쓰면 빠르겠다."**

열풍기 꺼내서 온도 450°C, 풍량 최대로 설정했다.

IC에 열풍 쐈더니 **IC가 풍압에 날아갔다.** 진짜로.

다시 올리고, 이번엔 풍량 줄였다. 온도는 그대로 450°C.

1분 지나도 납이 안 녹았다. **2분 지나니까 PCB가 휘기 시작했다.**

```
잘못된 설정:
┌────────────────────────────────────┐
│  온도: 450°C  → 너무 높음         │
│  풍량: 최대   → IC 날아감         │
│  시간: 2분+   → PCB 뒤틀림        │
└────────────────────────────────────┘
```

**올바른 설정을 찾기까지 PCB 2장 더 날렸다:**

```
올바른 설정:
┌────────────────────────────────────┐
│  온도: 350°C (Pb-free 기준)       │
│  풍량: 중간                        │
│  거리: 2~3cm                       │
│  시간: 30초~1분                    │
│  예열: 150°C로 30초 먼저          │
└────────────────────────────────────┘
```

## 네 번째 삽질: 부품 극성

IC 4개 다 붙였다. 수동 부품 납땜 시작.

**전해 커패시터 달았다. 전원 넣었다. '펑!'**

커패시터가 터졌다. 천장에 알루미늄 조각이 박혔다.

```
극성 반대로 달았다:

정방향:          역방향:
   +                 -
 ──┬──             ──┬──
   │                 │
 ┌─┴─┐             ┌─┴─┐
 │CAP│             │💥│
 └───┘             └───┘
   │                 │
 ──┴──             ──┴──
   -                 +
```

**전해 커패시터는 극성이 있다.** 실크 마킹이랑 실제 부품 마킹을 확인 안 했다.

그 뒤로 **극성 부품은 세 번 확인:**
1. 데이터시트에서 핀 배치
2. PCB 실크 마킹
3. 실제 부품 마킹

---

## 다섯 번째 삽질: 납땜 순서

PCB가 돌아왔다. 이번엔 제대로 하자.

**부품이 많은 쪽부터 납땜 시작했다.**

```
내가 한 순서:
1. 큰 커넥터 먼저
2. IC들
3. 수동 부품
```

IC 납땜하려는데 **커넥터가 방해해서 인두가 안 들어갔다.**

열풍기 쓰려니 **커넥터 플라스틱이 녹았다.**

```
올바른 순서:
┌────────────────────────────────────────────┐
│ 1. 가장 낮은 부품 (저항, 칩캡)             │
│ 2. IC (LQFP, QFN 등)                      │
│ 3. 중간 높이 부품 (전해캡, 크리스탈)       │
│ 4. 가장 높은 부품 (커넥터, 릴레이)         │
│ 5. 관통홀 부품 (있으면)                    │
└────────────────────────────────────────────┘
       ↑
   낮은 것부터 높은 것 순서
```

## 여섯 번째 삽질: 쇼트 찾기

납땜 완료. 테스터기로 전원 쇼트 체크.

**"삐익!" VCC-GND 쇼트.**

어디서 쇼트인지 몰랐다. 납땜한 곳이 200개가 넘었다.

**확대경으로 2시간 동안 찾았다.** 못 찾았다.

```
결국 찾은 방법:
1. 열화상 카메라 (없으면 손가락)
2. 전원 인가 (전류 제한!)
3. 발열 부위 탐색
```

쇼트 지점은 **0.1mm 납 찌꺼기 하나**였다. 맨눈으로는 절대 못 봤을 크기.

```
쇼트 찾기 팁:
┌────────────────────────────────────────────┐
│ 1. 전원 넣기 전 반드시 쇼트 체크!         │
│ 2. 전류 제한 전원 공급기 사용              │
│ 3. 발열 부위 = 쇼트 근처                   │
│ 4. 현미경 or 10배 확대경 필수             │
│ 5. 납땜 후 IPA로 세척 → 잔여물 제거       │
└────────────────────────────────────────────┘
```

## 일곱 번째 삽질: 콜드 조인트

쇼트 수정 후 전원 인가. **AD7280A 하나가 응답 안 했다.**

오실로스코프로 봤더니 신호가 끊겼다.

```
SPI 신호:

MCU → Dev0:  ┌┐┌┐┌┐┌┐ (정상)
Dev0 → Dev1: ┌┐┌┐┌┐┌┐ (정상)  
Dev1 → Dev2: ──────── (끊김!)
Dev2 → Dev3: ────────
```

Dev1 출력 핀에 테스터기 대봤더니 **도통이 안 됐다.**

핀을 확대경으로 봤더니 납땜은 되어 있었다. 근데 뭔가 이상했다.

**콜드 조인트(Cold Joint).**

```
정상 납땜:           콜드 조인트:
                     
   납이 흐름            납이 뭉침
   ∧∧∧               ○○○
  ─┴┴┴─             ─○○○─
                     표면만 붙음
                     내부 미접합
```

열이 부족하면 납이 녹기만 하고 접합이 안 된다. 겉보기엔 납땜된 것 같은데 실제로는 붙어있지 않다.

**해당 핀 다시 납땜.** 인두기 온도 확인하고, 핀과 패드를 **함께** 가열한 후 납 공급.

## 여덟 번째 삽질: 조립 순서

PCB 완성. 케이스에 조립.

커넥터 연결하고, 나사 조이고, 뚜껑 닫았다.

**"아, 점퍼 세팅 안 했다."**

뚜껑 열고, 커넥터 빼고, 점퍼 바꾸고, 다시 조립.

**"아, 펌웨어 안 넣었다."**

또 분해. 프로그래밍 헤더 연결. 다운로드. 다시 조립.

**"아, 시리얼 번호 스티커..."**

```
조립 체크리스트:
┌────────────────────────────────────────────┐

│ □ 쇼트 테스트 완료                         │

│ □ 펌웨어 다운로드 완료                     │

│ □ 캘리브레이션 완료                        │
│ □ 시리얼 번호 기록                         │

│ □ 커넥터 연결                              │
```

## 양산을 위한 준비

개발 단계에서 10장 만들었다. 이제 양산은 100장, 1000장.

**손으로 못 한다.**

```
개발:                양산:
                     
수작업 납땜          SMT 리플로우
2시간/장            5분/장
불량률 10%          불량률 0.1%
```

**SMT(Surface Mount Technology) 업체 선정:**

```
SMT 업체 체크리스트:
┌────────────────────────────────────────────┐
│ □ 0.5mm 피치 작업 가능?                   │
│ □ 0402 사이즈 실장 가능?                  │
│ □ BGA 리워크 가능?                        │
│ □ X-ray 검사 장비?                        │
│ □ AOI(자동광학검사) 있음?                 │
│ □ 최소 주문 수량?                         │
│ □ 리드타임?                               │
│ □ 부품 수급 대행?                         │
└────────────────────────────────────────────┘
```

## 정리: 제작/조립에서 배운 것들

| 삽질 | 증상 | 원인 | 해결 |
|------|------|------|------|
| 브릿지 | 핀 간 단락 | 납 과다, 플럭스 없음 | 플럭스 사용 |
| IC 날아감 | 부품 이동 | 열풍 과다 | 풍량 조절 |
| PCB 휨 | 뒤틀림 | 과열 | 예열, 온도 조절 |
| 커패시터 폭발 | 펑! | 극성 반대 | 세 번 확인 |
| 인두 접근 불가 | 납땜 불가 | 순서 잘못 | 낮→높 순서 |
| 원인 모를 쇼트 | VCC-GND | 납 찌꺼기 | 세척, 확대경 |
| 통신 끊김 | 부분 단선 | 콜드 조인트 | 충분한 가열 |
| 반복 분해 | 시간 낭비 | 체크리스트 없음 | 체크리스트! |

**가장 큰 교훈**: 납땜은 기술이다. 연습이 필요하다.
